# Contributing to this project

Thanks for your interest in contributing! There are several ways to help, whether you are a member of the project team or an outside contributor.

## Ways to contribute
- **Bug reports / feature requests:** open an issue: https://github.com/UBC-DSCI-310-2025W2/dsci-310-group-04/issues
- **Documentation:** improve README/report text/typos via a pull request.
- **Code/analysis:** submit changes via a pull request.
- **Reviews:** comment on open pull requests with constructive feedback.

## Workflow
- **Core team members:** create a branch from main, make focused commits, open a PR linked to an issue, get at least one review, then merge.
- **Outside contributors:** fork the repo, create a branch, and open a PR back to this repo.

We will review pull requests within 7 days.

## Code of Conduct
All contributors must follow our [Code of Conduct](CODE_OF_CONDUCT.md).
